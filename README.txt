# Basketball Rebound to Score Analyzer

## Опис

Знайти всі події, коли гравець у певному матчі після підбору (REBOUND) 
відразу здійснив кидок з гри (FIELD_GOAL_MADE), без жодних інших подій 
між ними. Асисти можна ігнорувати.

## Дата база

завдання працює лише з NBA Dataset

##  Вхідні дані

- `game_id` — ідентифікатор матчу
- Таблиці:
  - `play_records` — події у грі
  - `players` — інформація про гравців

## Вихідні атрибути

- `player_id`
- `first_name`
- `last_name`
- `period` (номер чверті)
- `period_time` (залишок часу у чверті)

## Сортування результатів

1. За `period` (зростання)
2. За `period_time` (спадання)
3. За `player_id` (зростання)
